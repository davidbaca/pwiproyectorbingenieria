Para este proyecto de web inicial, decidí realizar una sitio web para el trabajo de mi padre, por lo que es un sitio orientado a la ingeniería electrónica. 
Para el frontend utilizé Bootstrap con HTML y CSS. Diseñé el logo por mi cuenta. 
Del lado del servidor utilizé Node.js como se vieron en las últimas clases.
El formulario de contacto está conectado con Formspree para agilizar el contacto con los clientes y la base de datos está separada en el otro archivo db.txt del proyecto.
